# プロジェクト1：基本的なLambda関数の作成と実行

## 概要
このプロジェクトでは、AWS Lambdaを使用してサーバーレス関数を作成し、基本的なイベント駆動型処理を実装します。最初はシンプルな「Hello World」関数から始まり、最終的には複数のトリガーを持つ実用的な関数や、環境変数、レイヤーなどの高度な機能を使いこなせるようになります。

## 基本バージョン：シンプルなHello World関数

### 目的
Lambda関数を作成し、手動でテストイベントを実行して、基本的なサーバーレス処理の仕組みを理解する。

### 使われる概念
- **Lambda関数の作成**：コンソールまたはCLIを使用して関数を作成
- **ランタイムの選択**：Python、Node.js、Javaなど、使用する言語の選択
- **基本的なハンドラー関数**：イベントとコンテキストオブジェクトの理解
- **テストイベントの作成**：カスタムJSONペイロードで関数をテスト
- **CloudWatch Logsの確認**：関数の実行ログを確認して、デバッグの基礎を学ぶ
- **IAMロールの基礎**：Lambda実行ロールの自動作成と権限の概念
- **関数の実行時間とメモリ設定**：基本的なリソース配分の理解

### 学習ポイント
- サーバーレスアーキテクチャの基本概念
- イベント駆動型プログラミングモデル
- 関数のライフサイクル（コールドスタート、ウォームスタート）
- ログの出力と確認方法
- 基本的なエラーハンドリング

---

## 中級バージョン：API Gateway連携とデータ処理

### 目的
API Gatewayを使用してHTTPエンドポイントを作成し、Lambda関数をRESTful APIとして公開します。また、外部サービスとの連携やデータ変換処理を実装します。

### 追加される概念
- **API Gateway統合**：REST APIまたはHTTP APIを作成してLambdaと連携
- **HTTPリクエストの処理**：クエリパラメータ、パスパラメータ、リクエストボディの取得
- **レスポンスの整形**：適切なHTTPステータスコードとJSONレスポンスの返却
- **環境変数の活用**：設定値を関数コードから分離して管理
- **外部ライブラリの使用**：AWS SDKやサードパーティライブラリの追加
- **エラーハンドリングとリトライ**：適切な例外処理とリトライ戦略
- **タイムアウト設定**：関数の最大実行時間の調整
- **VPCアクセス基礎**：VPC内のリソースへのアクセス設定
- **CORS設定**：クロスオリジンリクエストへの対応

### 学習ポイント
- RESTful APIの設計原則
- HTTPプロトコルの理解
- 環境に応じた設定管理
- 外部依存関係の管理
- セキュリティベストプラクティス（APIキー、認証）
- パフォーマンスチューニングの基礎

## 発展バージョン：複数トリガー統合とCI/CD パイプライン

### 目的
S3、DynamoDB、SQS、EventBridgeなど複数のAWSサービスをトリガーとして統合し、Infrastructure as Code（IaC）を使用した自動デプロイメントパイプラインを構築します。

### 追加される概念
- **S3イベントトリガー**：ファイルアップロード時に自動実行
- **DynamoDB Streams**：テーブルの変更をトリガーとして処理
- **SQS統合**：キューからのメッセージを非同期処理
- **EventBridge（CloudWatch Events）**：スケジュール実行やカスタムイベントパターン
- **Lambda Layers**：共通ライブラリやコードの再利用
- **エイリアスとバージョン管理**：Blue/Greenデプロイメントの実現
- **SAM（Serverless Application Model）**：サーバーレスアプリケーションの定義とデプロイ
- **CloudFormation/Terraform**：インフラのコード化と自動化
- **デッドレターキュー（DLQ）**：失敗したイベントの処理
- **同時実行数の制御**：予約済み同時実行数と制限の設定
- **X-Rayによる分散トレーシング**：パフォーマンス分析とボトルネック特定
- **カスタムメトリクス**：CloudWatchカスタムメトリクスの送信
- **ストリーム処理の最適化**：バッチサイズとバッチウィンドウの調整
- **セキュリティの強化**：VPC設定、シークレット管理（Secrets Manager/Parameter Store）

### 学習ポイント
- イベント駆動型アーキテクチャの実践
- マイクロサービスパターンの実装
- Infrastructure as Code（IaC）の活用
- CI/CDパイプラインの構築
- 監視とトレーシングの実装
- コスト最適化戦略
- 本番環境での運用管理
- スケーラビリティとレジリエンスの設計
