## 調査結果

### 根本原因

Terraform の設定と Lambda コードのハンドラー名が一致していません：

| 項目 | Terraform設定 | 実際のコード |
|------|--------------|-------------|
| ハンドラー設定 | `index.handler` | 関数名は `lambda_handler` |

Terraform の main.tf (184行目) で：
```terraform
handler = "index.handler" # index.py の handler 関数
```

しかし、実際の Lambda コード（index.py）では：
```python
def lambda_handler(event, context):  # handler ではなく lambda_handler
```

### 解決方法

2つの選択肢があります：

**選択肢1**: Terraform の設定を修正する（推奨）
- `index.handler` → `index.lambda_handler` に変更

**選択肢2**: Lambda コードを修正する
- `def lambda_handler` → `def handler` に変更
