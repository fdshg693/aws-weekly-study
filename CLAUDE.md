# AWS勉強用レポジトリ

- 勉強課題
- AWSの知識メモ
- AWS関連のサンプルコード・プロジェクト

### ディレクトリ構成

- `.ai`
    - AI用のディレクトリ
- `.github`
    - Github Copilot設定等
- `aws_cli`
    - AWS CLI関連の設定・スクリプト
- `docs`
    - `docs/ai_generate`: AI生成ドキュメント
    - `docs/doc_templates`: ドキュメントテンプレート
    - `docs/learning_notes`: 学習メモ
    - `docs/memos`: 各種メモ
- `python`
    - Python関連のサンプルコード・プロジェクト
- `scripts`
    - 各種スクリプト
- `terra_*`
    - 各プロジェクト配下には、プロジェクト概要を示す`CLAUDE.md`を記述・更新すること
    - `terra_dev` : Terraformの実験用の、詳細なコメント付きサンプルコード
        - 各サブフォルダは独立したプロジェクトで、その下にフォルダがある場合は、そのプロジェクトのサブモジュールや関連コード
    - `terra_prod` : Terraformの実践的な必要最低限なプロジェクトコード
        - 各サブフォルダは独立したプロジェクトで、その下にフォルダがある場合は、そのプロジェクトのサブモジュールや関連コード
