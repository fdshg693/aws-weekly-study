# experimentsフォルダ配下のプロジェクト一覧

- `cognito`: Cognito・lambda・api gateway・dynamodb・amplifyを組み合わせた認証基盤の実装とセキュリティ対策の検証
- `discord_chatbot`: EC2を使って、メンションに反応してエコーレスポンスを返すDiscordチャットボット
- `ec2`: EC2+Ansible+Nginxを使って、簡単なWebサーバを構築
- `ecs`: ECS+Fargateを使って、簡単なWebサーバを構築
- `kendra_s3`: S3に保存されたドキュメントをKendraで検索するプロジェクト
- `kendra_web`: Webクローラーで収集したデータをKendraで検索するプロジェクト
- `lambda`
- `ollama_lambda_ec2`: EC2上のOllama（初期モデル: qwen2.5:0.5b）をLambda経由のAPIとして呼び出す構成
- `s3`: S3を使ったWEBサイトのホスティング
- `s3_cloudfront`: S3とCloudFrontを使ったWEBサイトのホスティング
- `sqs_lambda`
